# Boba — Chrome Extension

## Frontend Design Rules

Avoid generic "AI slop" aesthetics. Make creative, distinctive frontends that surprise and delight.

- **Typography**: Choose beautiful, unique fonts. Avoid generic fonts like Arial, Inter, Roboto, system fonts. The project uses Fredoka + Nunito — stay consistent with that pairing.
- **Color & Theme**: Commit to a cohesive aesthetic. Use CSS variables. Dominant colors with sharp accents outperform timid, evenly-distributed palettes. Boba uses a neobrutalist warm aesthetic (cream, brown borders, indigo accent, thick shadows).
- **Motion**: Use animations for effects and micro-interactions. CSS-only preferred. Focus on high-impact moments — staggered reveals over scattered micro-interactions.
- **Backgrounds**: Create atmosphere and depth. Layer CSS gradients, geometric patterns, contextual effects. Never default to solid colors.
- **Layouts**: Avoid predictable patterns and cookie-cutter components. Design for the specific context.

Avoid: overused font families, cliched purple-on-white gradients, predictable layouts, Space Grotesk, generic AI-generated aesthetics.
